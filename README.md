# PDF Merger

A robust Python desktop application for merging multiple PDF files into a single document with advanced control over the output.

## Features

- **User-Friendly GUI**: Clean and modern interface built with `tkinter` and `ttk`.
- **File Management**: Easily add, remove, and reorder PDF files before merging.
- **Detailed File Info**: View filename, file size, and page count for each added document.
- **Page Selection**: Choose specific pages from each PDF to include in the final merge.
- **Security & Permissions**:
  - Encrypt your merged PDF with a password.
  - Set user access permissions (Allow Printing, Editing, Copying).
- **Convenience**: Option to automatically open the merged file after saving.

## Requirements

- Python 3.x
- `pypdf` library

## Installation

1. Clone the repository.
2. Install the required dependencies:
   ```bash
   pip install pypdf
   ```

## Usage

Run the application using Python:

```bash
python merger.py
```
